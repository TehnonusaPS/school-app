# School App Project

Proyek ini dipisahkan menjadi dua bagian utama:
- `frontend/` : Berisi source code untuk aplikasi frontend (Vue/Vite/dll).
- `backend/`  : Berisi source code untuk aplikasi backend (saat ini masih dikosongkan).

## Cara Menjalankan

### Frontend
1. Masuk ke direktori frontend:
   ```bash
   cd frontend
   ```
2. Install dependencies (jika belum):
   ```bash
   npm install
   ```
3. Jalankan server development:
   ```bash
   npm run dev
   ```
